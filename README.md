# nam6.github.io

`nam6.github.io` 是 `nam6.com` 的内容博客与品牌展示站，使用 GitHub Pages + Jekyll 构建。

## 关于 nam6.com

`nam6.com` 聚焦全球主流社交媒体平台的推广与增长服务，服务方向覆盖 Instagram、TikTok、Facebook、YouTube、X、Telegram 等多平台场景。

这里更强调一站式协同体验：

1. 帮助品牌、跨境卖家、内容创作者和团队统一处理多平台推广需求
2. 提供粉丝增长、互动提升、播放曝光、评论活跃等常见服务方向
3. 重视下单效率、匿名提交、售后跟进与长期维护体验
4. 让内容传播、品牌展示与账号成长形成更顺畅的联动

## 这个仓库在做什么

这个仓库承载的是 `nam6.com` 的外部内容阵地，不只是站点源码，也承担品牌内容沉淀的角色。

主要用途如下：

1. 介绍 `nam6.com` 的业务方向与品牌定位
2. 输出多平台运营、内容增长和服务体验相关内容
3. 建立可长期更新的中文内容资产
4. 让主站业务入口与博客内容入口形成互相支持的双入口结构

## 站点内容方向

这里会持续围绕以下主题更新：

1. 多平台社交媒体增长观察
2. Instagram、TikTok、Facebook、YouTube、Telegram、X 等平台运营话题
3. 品牌推广、流量协同、内容传播与账号建设方法
4. 下单体验、售后支持、长期维护与站点建设笔记
