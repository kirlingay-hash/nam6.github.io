# nam6.github.io

`nam6.github.io` 是 `nam6.com` 的内容博客站，使用 GitHub Pages + Jekyll 部署。

## 已包含

1. 首页、404、关于、归档、分类、标签页
2. 文章详情页、上一篇/下一篇、相关文章、面包屑
3. RSS、sitemap、robots、SEO meta、结构化数据
4. 深色暖金品牌主题，区别于 `zfensi` 的蓝白极简风

## 写作方式

1. 在 `_posts/` 中新增文章，文件名格式：`YYYY-MM-DD-slug.md`
2. front matter 建议包含：`title`、`date`、`slug`、`tags`、`category`、`categories`、`description`、`cover`
3. 推送到 `main` 后由 GitHub Pages 自动部署
